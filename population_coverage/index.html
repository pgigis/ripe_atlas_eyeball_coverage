RIPE Atlas Population Coverage

Estimates the proportion of Internet users across the world situated on networks that we can measure from with RIPE Atlas.

The RIPE Atlas Population Coverage tool estimates the proportion of Internet users per country situated on networks that contain active RIPE Atlas probes. This is useful to:

  • Quickly assess the number of networks across the world that are reached by RIPE Atlas.
  • Determine where more RIPE Atlas probes are needed by spotting gaps in RIPE Atlas coverage at a glimpse.

Per-network user population data used by this tool is based on estimates provided by APNIC Labs.


The methodology can be found on the about page.
